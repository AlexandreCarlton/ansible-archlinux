# Firmware

Helps us stay up to date with our firmware.

To download the latest metadata on updates (optional - this role enables the
periodic refreshes):

```bash
fwupdmgr refresh
```

To list available updates on the system:

```bash
fwupdmgr refresh
```

To install these updates:

```
fwupdmgr update
```
